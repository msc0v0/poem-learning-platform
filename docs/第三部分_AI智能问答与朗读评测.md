# 第三部分：AI 智能问答与朗读核心模块及朗读评测

## 3.1 AI 智能问答系统架构

本系统的 AI 智能问答模块是古诗学习的核心功能，通过集成 Qwen2.5-7B-Instruct 大语言模型，为国际汉语学习者提供多语言、多层次、个性化的古诗词学习辅导。该模块实现了从基础字词解析到深层意境赏析的全方位智能问答服务。

### 3.1.1 多语言智能问答机制

系统支持**多语言提问**和**HSK 等级自适应回答**，这是针对国际汉语学习者的核心创新点。

**语言检测算法**：系统通过正则表达式和关键词匹配实现自动语言检测，支持中文、英语、西班牙语、法语、德语、日语、韩语等 7 种语言。检测逻辑基于字符比例和常见词汇分析：

```python
def detect_language(self, text: str) -> Tuple[str, float]:
    chinese_chars = len(re.findall(r'[\u4e00-\u9fff]', text))
    total_chars = len(text.strip())
    chinese_ratio = chinese_chars / total_chars
    
    if chinese_ratio > 0.3:
        return 'zh', chinese_ratio
    
    # 检测英语、西班牙语、法语等常见词
    # ...返回语言代码和置信度
```

**HSK 等级自适应**：系统根据用户的 HSK 等级（HSK1-HSK6）动态调整回答的复杂度和字数限制，确保学习内容与学习者的汉语水平相匹配。每个等级对应不同的参数配置：

| HSK 等级 | 最大字数 | 复杂度 | 适用场景 |
|---------|---------|--------|---------|
| HSK1 | 50字 | 极简单 | 初学者，使用基础词汇 |
| HSK2 | 80字 | 简单 | 入门级，简短句式 |
| HSK3 | 120字 | 中等 | 进阶级，常用表达 |
| HSK4 | 200字 | 中高级 | 中级学习者 |
| HSK5 | 300字 | 高级 | 高级学习者 |
| HSK6 | 500字 | 专家级 | 接近母语水平 |

**双语回答机制**：对于非中文提问，系统提供中文答案和母语翻译两个版本，帮助学习者理解的同时强化中文学习。翻译通过 Qwen 模型的多语言能力实现，确保翻译质量和文化准确性。

### 3.1.2 预设问答库（Common QA）

为提升响应速度和答案质量，系统内置了**常见问题库**，涵盖作者介绍、诗歌形式、文学术语等基础知识。当用户提问与预设问题匹配时，系统直接返回预设答案，响应时间从 2-5 秒降至毫秒级。

预设问答库包含以下类别：
- **作者介绍**：李白、杜甫、王维、孟浩然、白居易等著名诗人
- **诗歌形式**：五言绝句、七言绝句、律诗等体裁知识
- **文学术语**：意象、对仗、押韵等专业概念

问答匹配采用模糊匹配算法，支持同义词和近似表达，提高命中率。

### 3.1.3 四大问答类型

系统提供四种专业化的问答服务，每种类型对应不同的学习需求：

**1. 字词解析（Word Analysis）**

针对古诗中的生僻字词或关键字词，提供详细的语义解释、文化背景和使用示例。系统提示词设计如下：

```python
system_prompt = """你是一位专业的古诗词专家，专门为国际中文学习者提供古诗词字词解析。
请用简洁易懂的语言解释古诗中的字词含义，包括：
1. 字词的基本含义
2. 在诗中的具体含义
3. 相关的文化背景（如适用）
4. 使用例句帮助理解"""
```

**2. 创作背景解释（Background Explanation）**

介绍诗歌的创作时代背景、诗人生平、创作动机和历史情境，帮助学习者理解诗歌的深层含义。

**3. 意境赏析（Artistic Analysis）**

分析诗歌的意象、情感、艺术手法和美学价值，引导学习者感受古诗的艺术魅力。温度参数设置为 0.6，允许更具创造性和文学性的表达。

**4. 自由问答（General Question）**

支持任意形式的提问，系统根据诗歌上下文和用户问题生成个性化回答。

### 3.1.4 问答记录与质量评估

所有问答交互均保存至 `qa_records` 数据表，记录问题、答案、问题类型、响应时间等信息。系统支持用户对 AI 回答进行评分（1-5 星）和有用性反馈，用于持续优化问答质量。

## 3.2 语音朗读与发音评测系统

语音模块是系统的另一核心功能，通过 **TTS（文本转语音）** 和 **ASR（自动语音识别）** 技术，为学习者提供标准朗读示范和发音评估服务。

### 3.2.1 TTS 语音合成技术

**技术选型**：采用 **edge-tts 6.1.9**，基于微软 Edge 浏览器的免费 TTS 服务。选择理由：
1. **完全免费**：无需 API 密钥，降低运营成本
2. **高音质**：支持神经网络语音（Neural Voice），发音自然流畅
3. **多音色**：提供女声标准、男声标准、男声活力等多种音色
4. **语速可调**：支持慢速（-30%）、正常、快速（+30%）三档语速

**音色配置**：
```python
tts_voices = {
    'female_standard': 'zh-CN-XiaoxiaoNeural',  # 女声-标准（晓晓）
    'male_standard': 'zh-CN-YunxiNeural',       # 男声-标准（云希）
    'male_energetic': 'zh-CN-YunyangNeural',    # 男声-活力（云扬）
}
```

**异步生成流程**：TTS 音频生成采用异步编程（asyncio），避免阻塞主线程，提升用户体验。生成的音频文件以时间戳命名，存储在 `static/audio/` 目录，通过 URL 返回给前端播放。

### 3.2.2 ASR 语音识别技术

**技术选型**：采用 **OpenAI Whisper Tiny 模型**（39MB），这是一个轻量级的多语言语音识别模型。选择理由：
1. **模型小巧**：Tiny 版本仅 39MB，加载速度快，适合实时应用
2. **准确率高**：即使是最小版本，对中文识别准确率仍达 85% 以上
3. **离线运行**：模型可本地部署，无需外部 API 调用
4. **多语言支持**：自动检测语言，支持中英文混合识别

**懒加载机制**：为避免启动时的 DLL 依赖问题，Whisper 模型采用懒加载策略，仅在首次调用语音识别时才加载模型：

```python
def load_whisper_model(self, model_size='tiny'):
    if self.whisper_model is None:
        import whisper  # 延迟导入
        self.whisper_model = whisper.load_model(model_size)
    return self.whisper_model
```

### 3.2.3 发音评估算法

发音评估是系统的创新功能，通过多维度分析为学习者的朗读打分并提供改进建议。评估算法包含四个维度：

**1. 准确度评分（Accuracy Score）**

使用 **difflib.SequenceMatcher** 算法比对用户朗读文本与参考文本的相似度，计算公式：

```python
matcher = difflib.SequenceMatcher(None, reference_text, user_text)
accuracy_score = int(matcher.ratio() * 100)
```

同时识别错误类型：替换（replace）、遗漏（missing）、多余（extra），并标注错误位置。

**2. 流利度评分（Fluency Score）**

基于语速和停顿分析。理想语速设定为 **3-5 字/秒**，通过 librosa 库计算音频时长和字数：

```python
speech_rate = word_count / duration
if 3 <= speech_rate <= 5:
    rate_score = 100
```

停顿检测通过能量分析（RMS）实现，合理停顿次数为每 10 字 1-2 次。

**3. 音调评分（Tone/Intonation Score）**

使用 **librosa.piptrack** 提取音高序列，计算音高变化的标准差。理想的音高变化范围为 20-60 Hz，过于平淡或夸张都会降低分数：

```python
pitches, magnitudes = librosa.piptrack(y=audio, sr=sr)
pitch_std = np.std(pitch_sequence)

if 20 <= pitch_std <= 60:
    tone_score = 100
```

**4. 综合评分与星级**

综合评分采用加权平均：
- 准确度：50%
- 流利度：30%
- 音调：20%

```python
overall_score = int(accuracy * 0.5 + fluency * 0.3 + tone * 0.2)
```

星级评定：
- ⭐⭐⭐⭐⭐（90-100分）：优秀
- ⭐⭐⭐⭐（80-89分）：良好
- ⭐⭐⭐（70-79分）：及格
- ⭐⭐（60-69分）：需改进
- ⭐（<60分）：需大量练习

### 3.2.4 智能反馈生成

系统根据评分结果生成个性化反馈建议：

```python
if overall_score >= 90:
    feedback = "太棒了！您的发音非常标准！"
elif overall_score >= 80:
    feedback = "很好！继续保持，注意个别字词的发音。"
elif overall_score >= 70:
    feedback = "不错！多练习会更好，注意语调和流利度。"
# ...更多分级反馈
```

同时在前端高亮显示发音错误的字词，用不同颜色标注错误类型（红色=错误，黄色=遗漏），帮助学习者精准定位问题。

## 3.3 前端交互设计

### 3.3.1 AI 问答界面

采用**可调整宽度的分栏布局**，左侧显示古诗内容，右侧为 AI 对话面板。用户可通过拖拽分隔条调整两侧比例，适应不同屏幕和学习习惯。

问答类型按钮采用网格布局，提供快捷访问：
- 字词解析
- 创作背景
- 意境赏析
- 自由提问

对话历史以聊天气泡形式展示，用户消息居右（蓝色），AI 回答居左（灰色），支持滚动查看历史记录。

### 3.3.2 语音训练界面

**朗读播放区**：
- 音色选择下拉菜单（女声/男声）
- 语速调节按钮（慢速/正常/快速）
- 播放/暂停控制

**跟读录音区**：
- 录音按钮（点击开始/停止）
- 实时录音状态指示（红色圆点动画）
- 录音时长显示

**评分展示区**：
- 总分大字显示（带星级）
- 三维雷达图（准确度、流利度、音调）
- 分项分数条形图
- 字词级别的错误高亮
- 改进建议文本

## 3.4 技术亮点与创新
