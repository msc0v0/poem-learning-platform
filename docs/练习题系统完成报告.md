## 📚 题目示例

### 主旨理解题示例
```
题目：《静夜思》主要表达了诗人怎样的情感？

选项：
A. 思乡之情
B. 离别之苦
C. 登高望远之志
D. 隐居山林之意

正确答案：A

解析：《静夜思》通过描绘月光和诗人的所思所想，表达了诗人深深的思乡之情。诗中'举头望明月，低头思故乡'直接点明了主题。
```

### 字词理解题示例
```
题目：在《春晓》中，'不觉晓'的'晓'指的是什么？

选项：
A. 知道
B. 明白
C. 天亮
D. 晓谕

正确答案：C

解析：'晓'在这里的意思是天亮、黎明。整句'春眠不觉晓'的意思是春天的睡眠中不知不觉天就亮了。
```

### 意象赏析题示例
```
题目：《江雪》中'孤舟蓑笠翁，独钓寒江雪'描绘了怎样的意象？

选项：
A. 渔夫捕鱼的繁忙景象
B. 诗人孤独寂寞的心境
C. 冬日江面的美丽风光
D. 渔夫生活的艰辛

正确答案：B

解析：这两句诗通过'孤舟''独钓''寒江'等意象，描绘了一个孤独的渔翁在寒冷的江面上垂钓的场景，实际上是诗人借此表达自己孤独寂寞、不为世俗所理解的心境。
```

## 📝 技术实现

### 核心技术栈
- **AI模型**: Qwen2.5-7B-Instruct
- **API平台**: 硅基流动 (SiliconFlow)
- **后端框架**: Flask + SQLAlchemy
- **数据库**: SQLite
- **前端**: HTML + TailwindCSS + JavaScript

### 数据模型
```python
Exercise (练习题)
├── id - 题目ID
├── poem_id - 古诗ID
├── question_type - 题型
├── question - 题目内容
├── options - 选项（JSON）
├── correct_answer - 正确答案
├── explanation - 答案解析
├── difficulty - 难度等级
└── points - 分值

UserExerciseRecord (答题记录)
├── id - 记录ID
├── user_id - 用户ID
├── exercise_id - 题目ID
├── user_answer - 用户答案
├── is_correct - 是否正确
├── score - 得分
├── time_spent - 答题用时
└── answered_at - 答题时间
```
