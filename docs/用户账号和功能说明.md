## 📝 注册功能

### 前端注册界面
- **位置**: 古诗列表页面 `/poems`
- **入口**: 点击导航栏右上角的 "注册" 按钮
- **功能**: 
  - 填写用户名、邮箱、密码
  - 密码确认验证
  - 邮箱格式验证
  - 自动登录

### 注册API
```
POST /api/auth/register
Content-Type: application/json

{
  "username": "新用户名",
  "email": "email@example.com",
  "password": "密码(至少6位)"
}
```

## 💾 学习数据存储功能

平台能够完整记录和存储以下用户学习数据：

### 1. 学习记录 (LearningRecord)
存储在 `learning_records` 表中：
- ✅ **阅读时长** - 用户阅读古诗的时间（秒）
- ✅ **视频观看时长** - 意境视频观看时间
- ✅ **音频收听时长** - 朗读音频收听时间
- ✅ **字词查询记录** - JSON格式存储查询的字词
- ✅ **AI交互次数** - 与AI助手的对话次数
- ✅ **学习完成度** - 0-1的完成率
- ✅ **首次访问/最后访问时间**

### 2. 练习成绩记录 (UserExerciseRecord)
存储在 `user_exercise_records` 表中：
- ✅ **用户答案** - 每题的作答内容
- ✅ **正确性判断** - 答题是否正确
- ✅ **得分** - 本题获得的分数
- ✅ **答题用时** - 每题花费的时间（秒）
- ✅ **答题时间戳** - 精确的答题时间

### 3. 注意力追踪 (AttentionTracking) 🆕
存储在 `attention_tracking` 表中：
- ✅ **鼠标移动轨迹** - JSON格式记录鼠标坐标和时间戳
- ✅ **鼠标点击记录** - 点击位置、元素、时间
- ✅ **滚动事件** - 页面滚动方向和位置
- ✅ **鼠标停留区域** - 热力图数据
- ✅ **阅读模式分析** - 阅读行为模式
- ✅ **专注时长** - 集中注意力的时间
- ✅ **分心次数** - 离开页面或切换标签的次数
- ✅ **各区域停留时间** - 古诗区/AI对话区/练习区的时间分配
- ✅ **交互密度评分** - 用户交互活跃度
- ✅ **参与度评分** - 综合学习投入度评分

### 4. AI问答记录 (QARecord)
存储在 `qa_records` 表中：
- ✅ **问题内容** - 用户提出的问题
- ✅ **AI回答** - AI助手的回复
- ✅ **问题类型** - 字词解析/背景介绍/意境赏析/其他
- ✅ **响应时间** - AI回答所需时间
- ✅ **用户评分** - 对AI回答的评价
- ✅ **是否有帮助** - 用户反馈

## 🔌 注意力追踪API

### 记录注意力数据
```
POST /api/attention/track
Authorization: Bearer <token>
Content-Type: application/json

{
  "poem_id": 1,
  "session_id": "unique-session-id",
  "mouse_movements": [{x: 100, y: 200, timestamp: 1234567890}, ...],
  "mouse_clicks": [{x: 100, y: 200, element: "poem-title", timestamp: 1234567890}, ...],
  "scroll_events": [{position: 500, direction: "down", timestamp: 1234567890}, ...],
  "focus_duration": 180,
  "distraction_count": 2,
  "time_on_poem": 120,
  "time_on_ai_chat": 60,
  "interaction_density": 0.85,
  "engagement_score": 0.92,
  "session_ended": false
}
```

### 获取注意力统计
```
GET /api/attention/statistics?poem_id=1
Authorization: Bearer <token>
```

返回：
```json
{
  "total_sessions": 5,
  "average_focus_duration": 180,
  "average_engagement_score": 0.88,
  "total_distractions": 8,
  "sessions": [...]
}
```

## 📊 数据分析能力

平台可以基于存储的数据进行以下分析：

1. **学习行为分析**
   - 学习时长趋势
   - 学习频率模式
   - 难度偏好分析

2. **注意力分析**
   - 专注度评估
   - 阅读模式识别
   - 交互热点分析

3. **学习效果评估**
   - 练习正确率统计
   - 不同题型表现
   - 知识点掌握情况

4. **个性化推荐**
   - 基于学习记录推荐古诗
   - 基于薄弱环节推荐练习
   - 基于注意力模式优化内容展示

## 🚀 快速开始

1. **启动服务器**
   ```bash
   python app.py
   ```

2. **访问平台**
   ```
   http://localhost:5000/poems
   ```

3. **注册/登录**
   - 点击右上角"注册"或"登录"按钮
   - 使用测试账号或创建新账号

4. **开始学习**
   - 选择古诗卡片进入学习页面
   - 使用AI助手提问
   - 完成配套练习
   - 系统自动记录所有学习数据

## 📝 注意事项

- 注意力追踪需要用户登录后才能记录
- 所有学习数据与用户账号绑定
- 数据存储采用JSON格式便于后续分析
- 支持实时追踪和批量上传两种模式
