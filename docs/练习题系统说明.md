# 📝 练习题系统使用说明

## 🎯 功能概述

古诗练习题系统使用**硅基流动API (Qwen大模型)**自动生成高质量的练习题，帮助学生巩固学习成果。

### 题型设置
每首古诗配备 **3道练习题**，涵盖以下类型：

| 题型 | 说明 | 考查重点 |
|------|------|----------|
| **主旨理解** | 考查对诗歌中心思想和主题的把握 | 整体理解能力 |
| **字词理解** | 考查对诗中关键字词的理解 | 词汇掌握程度 |
| **意象赏析** | 考查对诗歌意象和艺术手法的理解 | 鉴赏分析能力 |

## 🚀 快速开始

### 1. 批量生成练习题

**方式一：使用Python脚本**
```bash
python generate_exercises.py
```

**方式二：使用API接口**
```bash
curl -X POST http://localhost:5000/api/exercises/batch-generate \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json"
```

### 2. 为单首诗生成练习题

```bash
curl -X POST http://localhost:5000/api/exercises/generate/1 \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"question_type": "主旨理解"}'
```

### 3. 前端使用

1. 访问古诗学习页面：`http://localhost:5000/poem/{poem_id}`
2. 点击**"练习题"**标签
3. 如果没有题目，点击**"生成练习题"**按钮
4. 选择答案并提交
5. 查看答案解析和成绩统计

## 💡 题目生成说明

### AI生成流程

1. **分析古诗内容**
   - 解析诗歌标题、作者、内容
   - 理解诗歌主题和意境

2. **生成题目**
   - 根据题型要求生成问题
   - 创建4个选项（1个正确答案 + 3个干扰项）
   - 编写详细的答案解析

3. **质量保证**
   - 选项具有一定迷惑性
   - 难度适合国际中文学习者
   - 解析清晰易懂

### 题目格式

```json
{
  "question": "这首诗主要表达了什么情感？",
  "options": [
    "A. 思乡之情",
    "B. 报国之志", 
    "C. 离别之苦",
    "D. 山水之乐"
  ],
  "correct_answer": "A",
  "explanation": "这首诗通过描写..., 表达了诗人深深的思乡之情。诗中'举头望明月，低头思故乡'直接点明了主题。"
}
```

## ⚙️ 配置说明

### 环境变量
```bash
# .env文件
QWEN_API_KEY=your_api_key_here
QWEN_API_URL=https://api.siliconflow.cn/v1/chat/completions
```

### 参数配置

**QwenService 参数**:
- `model`: "Qwen/Qwen2.5-7B-Instruct"
- `temperature`: 0.3 (生成稳定性)
- `max_tokens`: 800 (单题生成)

**Exercise 模型**:
- `difficulty`: 1-5 (默认2)
- `points`: 每题10分
- `question_type`: 主旨理解/字词理解/意象赏析

## 🔧 故障排除

### 常见问题

**1. 生成失败**
- 检查API密钥是否正确
- 确认网络连接正常
- 查看API调用次数限制

**2. JSON解析错误**
- AI返回格式可能不标准
- 脚本会自动清理markdown代码块
- 查看日志中的原始返回内容

**3. 题目数量不足**
- 部分古诗可能生成失败
- 可手动重新生成
- 检查失败日志
