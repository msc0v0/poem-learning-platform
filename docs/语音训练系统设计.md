# 智能跟读与语音训练系统设计

## 📋 功能概述

为古诗学习平台添加智能跟读和语音训练功能，帮助国际中文学习者提高发音准确度和朗读流利度。

---

## ✨ 核心功能

### 1. 朗读功能（增强版）

**当前状态**：
- ✅ 已实现基础TTS（使用浏览器Web Speech API）
- ❌ 朗读按钮重复（需要去重）
- ❌ 没有语速和声音选择

**需要改进**：
- [x] 去除重复的朗读按钮
- [ ] 添加语速控制（慢速/正常/快速）
- [ ] 添加声音选择（男声/女声）
- [ ] 添加分句朗读功能
- [ ] 可视化朗读进度（高亮当前朗读的句子）

### 2. 跟读功能（新增）

**核心流程**：
```
1. 学生点击"开始跟读"
2. 系统播放标准音频（一句）
3. 学生录音模仿
4. 系统评分并反馈
5. 重复直到完成全诗
```

**评分维度**：
- **准确度（Accuracy）**：字词发音是否准确（0-100分）
- **流利度（Fluency）**：朗读是否流畅，有无停顿（0-100分）
- **完整度（Completeness）**：是否读完整（0-100分）
- **语调（Prosody）**：声调是否准确（0-100分）

**反馈方式**：
- 总分展示（如：85/100或⭐⭐⭐⭐☆）
- 发音不准的字词高亮标注（红色）
- 提供改进建议

### 3. 挑战模式（新增）

**规则**：
- 逐句跟读，每句需达到60分才能进入下一句
- 全部句子通过后获得勋章
- 记录最佳成绩和通关时间

**勋章系统**：
- 🥉 铜牌：60-75分
- 🥈 银牌：76-85分
- 🥇 金牌：86-95分
- 💎 钻石：96-100分

## 🛠️ 技术方案

### 方案A：云服务API（推荐）

#### 优点
- ✅ 准确度高
- ✅ 支持专业发音评分
- ✅ 无需本地部署模型
- ✅ 可扩展性强

#### 缺点
- ❌ 需要API费用
- ❌ 依赖网络

#### 推荐服务商

**1. 阿里云智能语音交互**
- API: [语音评测](https://help.aliyun.com/document_detail/92133.html)
- 价格: ¥0.02/次（前100万次免费）
- 优势: 专门支持中文发音评分，效果好

**2. 腾讯云语音识别**
- API: [语音评测](https://cloud.tencent.com/document/product/884/19322)
- 价格: ¥0.025/次
- 优势: 中文支持好

**3. Microsoft Azure Speech**
- API: [Pronunciation Assessment](https://learn.microsoft.com/zh-cn/azure/ai-services/speech-service/how-to-pronunciation-assessment)
- 价格: $1/1000次
- 优势: 国际化好，支持多语言

### 方案B：开源模型（备选）

**模型选择**：
- `whisper`（OpenAI）：语音识别
- `WeTextProcessing`：文本规范化
- 自研评分算法

#### 优点
- ✅ 免费
- ✅ 隐私性好

#### 缺点
- ❌ 评分准确度较低
- ❌ 需要GPU资源
- ❌ 开发成本高

## 💾 数据库设计

### 1. 语音评测记录表 (speech_evaluations)

```sql
CREATE TABLE speech_evaluations (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    user_id INTEGER NOT NULL,
    poem_id INTEGER NOT NULL,
    sentence_index INTEGER,  -- 第几句（0-based）
    sentence_text TEXT,      -- 句子内容
    
    -- 录音信息
    audio_url TEXT,          -- 录音文件URL
    audio_duration FLOAT,    -- 录音时长（秒）
    
    -- 评分结果
    accuracy_score FLOAT,    -- 准确度 0-100
    fluency_score FLOAT,     -- 流利度 0-100
    completeness_score FLOAT,-- 完整度 0-100
    prosody_score FLOAT,     -- 语调 0-100
    overall_score FLOAT,     -- 总分 0-100
    
    -- 详细反馈
    problem_words JSON,      -- 发音有问题的字词 [{word: "床", score: 45}]
    suggestions TEXT,        -- 改进建议
    
    -- 元数据
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    
    FOREIGN KEY (user_id) REFERENCES users(id),
    FOREIGN KEY (poem_id) REFERENCES poems(id)
);
```

### 2. 朗读挑战表 (reading_challenges)

```sql
CREATE TABLE reading_challenges (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    user_id INTEGER NOT NULL,
    poem_id INTEGER NOT NULL,
    
    -- 挑战状态
    status TEXT,             -- 'in_progress', 'completed', 'abandoned'
    current_sentence INTEGER DEFAULT 0,
    total_sentences INTEGER,
    
    -- 成绩
    total_score FLOAT,       -- 总平均分
    best_score FLOAT,        -- 历史最佳分
    completion_time INTEGER, -- 完成时间（秒）
    
    -- 勋章
    badge_earned TEXT,       -- 'bronze', 'silver', 'gold', 'diamond'
    
    -- 时间戳
    started_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    completed_at DATETIME,
    
    FOREIGN KEY (user_id) REFERENCES users(id),
    FOREIGN KEY (poem_id) REFERENCES poems(id)
);
```

### 3. 勋章表 (badges)

```sql
CREATE TABLE badges (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    user_id INTEGER NOT NULL,
    badge_type TEXT,         -- 'pronunciation_master', 'perfect_reader', etc.
    badge_level TEXT,        -- 'bronze', 'silver', 'gold', 'diamond'
    poem_id INTEGER,         -- 如果是针对某首诗的勋章
    earned_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    
    FOREIGN KEY (user_id) REFERENCES users(id),
    FOREIGN KEY (poem_id) REFERENCES poems(id)
);
```

## 📝 API设计

### 1. 文本转语音

```http
POST /api/speech/tts
Content-Type: application/json
Authorization: Bearer <token>

{
  "text": "床前明月光，疑是地上霜。",
  "voice": "female",  // 'male' | 'female'
  "speed": "normal"   // 'slow' | 'normal' | 'fast'
}

Response:
{
  "success": true,
  "audio_url": "/static/audio/temp_xxx.mp3",
  "duration": 5.2
}
```

### 2. 评测发音

```http
POST /api/speech/evaluate
Content-Type: multipart/form-data
Authorization: Bearer <token>

{
  "audio": <File>,
  "poem_id": 1,
  "sentence_index": 0,
  "reference_text": "床前明月光"
}

Response:
{
  "success": true,
  "evaluation": {
    "accuracy_score": 85.5,
    "fluency_score": 90.0,
    "completeness_score": 95.0,
    "prosody_score": 80.0,
    "overall_score": 87.6,
    "problem_words": [
      {
        "word": "霜",
        "score": 45,
        "expected_pinyin": "shuāng",
        "actual_pinyin": "suāng"
      }
    ],
    "suggestions": "注意'霜'的发音，需要发sh音而不是s音"
  },
  "evaluation_id": 123
}
```

### 3. 挑战模式

```http
# 开始挑战
POST /api/speech/challenge/start
{
  "poem_id": 1
}

Response:
{
  "challenge_id": 456,
  "total_sentences": 4,
  "sentences": ["床前明月光", "疑是地上霜", ...]
}

# 提交一句
POST /api/speech/challenge/submit
{
  "challenge_id": 456,
  "sentence_index": 0,
  "audio": <File>
}

Response:
{
  "passed": true,
  "score": 85,
  "next_sentence_index": 1,
  "challenge_completed": false
}

# 完成挑战
GET /api/speech/challenge/:id/result

Response:
{
  "challenge_id": 456,
  "status": "completed",
  "total_score": 88.5,
  "badge_earned": "gold",
  "completion_time": 320
}
```

## 💰 成本估算

### 阿里云智能语音（推荐方案）

- **免费额度**：前100万次调用免费
- **付费价格**：¥0.02/次
- **预估用量**：
  - 每个用户每天练习5首诗 × 4句 = 20次
  - 100个活跃用户 = 2000次/天
  - 月成本：2000 × 30 × ¥0.02 = ¥1200

**建议**：先使用免费额度测试，后续根据用户量决定是否升级。
